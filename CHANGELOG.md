# 更新日志

## v2.2.0 (2026-06-23) - LLM 意图路由系统

### ✨ 新增功能

#### LLM 意图路由器 (`agent/intent_router.py`)
- **自然语言理解**：完全基于 LLM 理解用户消息，替代传统关键词匹配
- **智能意图识别**：支持 7 种意图类型（巡检、PPT、资产、课表、搜索、知识库、对话）
- **参数自动提取**：从自然语言中自动提取时间、人员、操作等参数
- **上下文理解**：支持理解指代和上下文关系
- **降级机制**：LLM 失败时自动降级到关键词匹配

#### 意图监控系统 (`agent/intent_monitor.py`)
- **性能监控**：记录每次识别的延迟、成功率
- **准确性分析**：统计意图分布、置信度分布
- **错误追踪**：记录失败的识别请求
- **慢调用检测**：识别耗时过长的请求

#### 缓存机制
- **LRU 缓存**：最多缓存 1000 条常见意图
- **TTL 过期**：缓存有效期 5 分钟
- **自动清理**：过期自动删除

#### API 端点 (`web/intent_api.py`)
- `/api/intent/classify` - 识别用户意图
- `/api/intent/stats` - 获取统计信息
- `/api/intent/recent` - 获取最近记录
- `/api/intent/errors` - 获取错误记录
- `/api/intent/slow` - 获取慢调用记录
- `/api/intent/hourly` - 获取每小时统计

### 🔧 优化改进

#### main.py 集成
- 在技能匹配前添加 LLM 意图识别
- 将识别的意图传递给技能执行
- 保持向后兼容：LLM 未匹配时回退到原有规则

#### 技能系统增强
- 技能接收 `intent` 参数，支持动态参数传递
- 巡检技能支持根据时间参数查询不同日期的数据
- 资产技能支持根据操作类型执行不同操作

## v2.1.0 (2026-05-26) - 搜索功能

### ✨ 新增功能

#### 网络搜索模块
- **通用搜索**：支持搜索任意关键词
- **教学资源搜索**：支持搜索课件、教案、习题、素材等
- **新闻查询**：支持查询最新信息和新闻
- **搜索增强回答**：知识性问题自动结合搜索结果回答

#### 搜索功能特点
- 多源搜索：百度、必应等多个搜索源
- 智能缓存：搜索结果缓存5分钟，提高响应速度
- 结果格式化：搜索结果以清晰的格式展示
- 异步处理：搜索过程不阻塞主线程

### 🔧 优化改进

#### 消息处理优化
- 添加消息去重机制，避免重复处理
- 优化回复逻辑，减少重复消息
- 改进错误处理，提高系统稳定性

#### PPT生成优化
- 改进SVG页面设计，更加专业美观
- 优化内容生成逻辑，确保内容完整
- 添加内容验证，防止空白页面

### 📝 使用示例
