# PPT Master 集成使用说明

## 功能介绍

现在钉钉机器人已经集成了 **PPT Master** 专业PPT生成能力，可以生成高质量、可编辑的教学课件！

## 使用方法

### 方式1：自动生成（推荐）

直接发送包含关键词的消息，系统会自动选择最佳生成方式：

```
请为高中数学《导数的概念》生成教案
请为初中英语《一般过去时》生成课件
请为小学语文《草船借箭》生成说课稿
请为高中物理《牛顿第二定律》生成教学反思
```

### 方式2：使用PPT Master生成高质量课件

在消息中添加"专业"或"高质量"关键词，会使用PPT Master生成更精美的课件：

```
请为高中数学《导数的概念》生成专业课件
请为初中英语《一般过去时》生成高质量教案
请为小学语文《草船借箭》生成专业说课稿
使用ppt-master为高中物理《牛顿第二定律》生成教学反思
```

## 支持的关键词

### 内容类型关键词
- **教案** - 完整的教学设计方案
- **课件** - 教学课件大纲
- **说课稿** - 说课演讲稿
- **反思** - 教学反思分析

### 难度级别关键词
- **基础** - 面向学习困难学生，注重基础概念
- **中等** - 面向中等水平学生（默认）
- **提高** - 面向学有余力学生，注重思维拓展
- **拓展** - 面向学科特长生，注重创新思维

### 质量级别关键词
- **专业** - 使用PPT Master生成专业级课件
- **高质量** - 使用PPT Master生成高质量课件
- **ppt-master** - 明确指定使用PPT Master
- **大师级** - 使用PPT Master生成大师级课件

## 示例

### 示例1：基础教案生成
```
请为高中数学《导数的概念》生成教案，难度中等
```
**输出：** 标准教案PPT，包含教学目标、重难点、教学过程等

### 示例2：专业课件生成
```
请为初中英语《一般过去时》生成专业课件
```
**输出：** 使用PPT Master生成的专业课件，设计精美，可在PowerPoint中编辑

### 示例3：指定学情
```
请为小学数学《分数的初步认识》生成教案
学情：学生是三年级学生，刚接触分数概念
难度：基础
```
**输出：** 根据学情调整的基础难度教案

### 示例4：完整信息
```
请为高中化学《氧化还原反应》生成高质量课件
学科：化学
年级：高一
难度：提高
风格：学术风格
```
**输出：** 学术风格的高质量化学课件

## 生成流程

### 标准流程（自动）
1. 识别关键词和需求
2. 调用AI生成内容大纲
3. 生成教育PPT
4. 发送文件

### PPT Master流程（专业）
1. 识别"专业"或"高质量"关键词
2. 解析学科、年级、难度等信息
3. 调用PPT Master创建项目
4. 生成设计规范
5. 生成SVG页面
6. 转换为PPTX
7. 发送专业级课件

## 技术说明

### PPT Master优势
- **原生可编辑** - 每个元素都能在PowerPoint中点击修改
- **专业设计** - 遵循设计规范，视觉效果专业
- **多种风格** - 支持教育、学术、简约等多种风格
- **高质量输出** - 使用SVG技术，保证清晰度

### 文件格式
- 输出格式：PPTX（PowerPoint格式）
- 兼容性：支持Microsoft PowerPoint、WPS等
- 编辑性：所有元素均可编辑

## 常见问题

### Q1: 为什么有时候会回退到基础模式？
A: PPT Master生成需要更多时间和资源，如果遇到网络问题或API限制，系统会自动回退到基础模式，确保用户能收到课件。

### Q2: 如何获得最好的效果？
A: 建议：
1. 明确指定学科和年级
2. 提供详细的学情信息
3. 使用"专业"或"高质量"关键词
4. 指定合适的难度级别

### Q3: 生成时间需要多久？
A:
- 基础模式：约30秒-1分钟
- PPT Master模式：约1-2分钟

### Q4: 支持哪些学科？
A: 支持所有中小学学科，包括：
- 语文、数学、英语
- 物理、化学、生物
- 历史、地理、政治
- 音乐、美术、体育等

### Q5: 如何自定义课件风格？
A: 在消息中指定风格要求，例如：
```
请为高中数学《导数的概念》生成专业课件，风格：简约现代
```
