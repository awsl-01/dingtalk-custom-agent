# 教育PPT生成系统

## 功能概述

教育PPT生成系统是一个基于AI的智能教学材料生成工具，支持自动生成以下教学材料：

1. **教案生成** - 完整的教学设计方案
2. **课件大纲** - 课件内容框架和结构
3. **说课稿** - 说课演讲稿
4. **教学反思** - 多维度教学反思分析

## 核心特性

### 🎯 智能内容生成
- 基于教材章节自动生成教学内容
- 支持多学科、多年级
- 内容结构化、专业化

### 📊 学情分析与难度调整
- **基础难度**：面向学习困难学生，注重基础概念和基本技能
- **中等难度**：面向中等水平学生，兼顾基础与提高
- **提高难度**：面向学有余力学生，注重思维拓展
- **拓展难度**：面向学科特长生，注重创新思维培养

### 🎨 专业PPT设计
- 现代化教育主题设计
- 清晰的视觉层次
- 专业的配色方案

## 使用方法

### 1. 通过钉钉机器人使用

直接在钉钉群聊或私聊中发送消息：

```
# 生成教案
请为高中数学《导数的概念》生成教案，难度中等

# 生成课件大纲
请为初中英语《一般过去时》生成课件大纲

# 生成说课稿
请为小学语文《草船借箭》生成说课稿

# 生成教学反思
请为高中物理《牛顿第二定律》生成教学反思

# 指定学情
请为小学数学《分数的初步认识》生成教案
学情：学生是三年级学生，刚接触分数概念
难度：基础
```

### 2. 通过Python代码使用

```python
from agent.education_ppt_generator import generate_education_ppt

# 基础用法
path, title = generate_education_ppt(
    "请为高中数学《导数的概念》生成教案，难度中等"
)
print(f"生成完成：{path}")

# 指定输出目录
path, title = generate_education_ppt(
    "请为初中英语《一般过去时》生成课件大纲",
    output_dir="my_output"
)
```

### 3. 分步骤生成（高级用法）

```python
from agent.education_ppt_generator import (
    generate_lesson_plan,
    generate_slide_outline,
    generate_lesson_speech,
    generate_teaching_reflection,
    create_lesson_plan_ppt,
    create_slide_outline_ppt,
    create_lesson_speech_ppt,
    create_teaching_reflection_ppt
)

# 步骤1：生成教案数据
lesson_plan = generate_lesson_plan(
    "高中化学《氧化还原反应》",
    "提高",
    "学生是重点班学生，化学基础扎实"
)

# 步骤2：基于教案生成其他材料
slide_outline = generate_slide_outline(lesson_plan, "学术")
lesson_speech = generate_lesson_speech(lesson_plan)
teaching_reflection = generate_teaching_reflection(lesson_plan)

# 步骤3：创建PPT文件
create_lesson_plan_ppt(lesson_plan, "教案.pptx")
create_slide_outline_ppt(slide_outline, "课件大纲.pptx")
create_lesson_speech_ppt(lesson_speech, "说课稿.pptx")
create_teaching_reflection_ppt(teaching_reflection, "教学反思.pptx")
```

## 教案结构

生成的教案包含以下完整结构：

```
1. 基本信息
   - 标题、章节、学科、年级、课时、难度

2. 教学目标
   - 知识与技能目标
   - 过程与方法目标
   - 情感态度价值观目标

3. 教学重难点
   - 教学重点
   - 教学难点

4. 学情分析
   - 学情水平
   - 前置知识
   - 学生兴趣点

5. 教学过程
   - 课堂导入（5分钟）
   - 新课讲授（25分钟）
   - 巩固练习（10分钟）
   - 课堂小结（3分钟）
   - 作业布置

6. 板书设计
   - 板书标题
   - 板书结构

7. 教学反思
   - 预期亮点
   - 可能问题
   - 改进方向
```

## 课件大纲结构

生成的课件大纲包含：

```
1. 封面页
2. 目录页
3. 教学目标页
4. 课堂导入页
5. 知识讲解页（多页）
6. 例题演示页
7. 课堂练习页
8. 知识总结页
9. 作业布置页
10. 结束页
```

## 说课稿结构

生成的说课稿包含：

```
一、说教材
二、说学情
三、说教学目标
四、说教学重难点
五、说教法与学法
六、说教学过程
   （一）导入新课
   （二）探究新知
   （三）巩固练习
   （四）课堂小结
   （五）布置作业
七、说板书设计
八、说教学反思
```

## 教学反思结构

生成的教学反思包含：

```
一、教学目标达成情况
二、教学内容处理
三、教学方法运用
四、学生参与度分析
五、课堂生成处理
六、教学效果评估

反思总结与改进计划
```

## 常见问题

### Q1: 生成的内容不够详细怎么办？
A: 可以在输入中添加更多具体要求，例如：
- "需要详细的教学步骤"
- "每个环节要有明确的时间分配"
- "增加学生活动设计"

### Q2: 如何调整难度？
A: 在输入中明确指定难度级别：
- "难度：基础" - 适合学习困难学生
- "难度：中等" - 适合普通学生（默认）
- "难度：提高" - 适合学有余力学生
- "难度：拓展" - 适合学科特长生

### Q3: 支持哪些学科？
A: 理论上支持所有学科，包括：
- 语文、数学、英语
- 物理、化学、生物
- 历史、地理、政治
- 音乐、美术、体育
- 信息技术等

### Q4: 如何自定义课件风格？
A: 在生成课件大纲时可以指定风格：
- "风格：简约" - 简洁大方
- "风格：活泼" - 色彩丰富
- "风格：学术" - 严谨专业
- "风格：科技" - 现代科技感
