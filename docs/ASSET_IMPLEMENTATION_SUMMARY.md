## 三、功能特性

### 3.1 核心功能

| 功能 | 说明 | 示例 |
|------|------|------|
| 资产录入 | 添加新资产信息 | `录入资产 投影仪 教学设备 301教室` |
| 资产查询 | 按名称/编号查询 | `查询资产 投影仪` |
| 资产借用 | 记录借用信息 | `借用资产 AST20260618001 李老师` |
| 资产归还 | 记录归还信息 | `归还资产 AST20260618001` |
| 资产统计 | 生成统计报告 | `资产统计` |
| 资产录入模板 | 下载批量导入模板 | `资产录入模板` |
| 资产盘点模板 | 下载盘点模板 | `资产盘点模板` |
| 批量导入资产 | 批量导入资产数据 | `批量导入资产`（需先上传模板） |
| 帮助信息 | 查看使用指南 | `资产管理` |

### 3.2 技术特性

- **自动加载**：新技能自动注册，无需修改 `main.py`
- **数据隔离**：按 `corp_id` 隔离不同学校的数据
- **意图识别**：支持多种表述方式，关键词匹配
- **编号生成**：AST + 日期 + 序号，自动递增
- **状态管理**：在用 → 借用中 → 在用，完整生命周期

### 3.3 支持的资产分类

- 教学设备：投影仪、电脑、白板、音响、麦克风
- 办公设备：打印机、复印机、扫描仪
- 家具：桌椅、课桌、椅子、柜子
- 教材：教材、课本、教辅
- 其他：未分类资产

## 七、部署说明

### 7.1 部署步骤

1. **无需额外安装依赖**
   - 资产管理功能使用 Python 标准库（json, os, re, datetime）

2. **重启服务**
   ```bash
   python main.py
   # 或
   start.bat
   ```

3. **验证部署**
   ```bash
   # 运行测试脚本
   python tests/test_asset.py

   # 或快速测试
   python tests/quick_asset_test.py
   ```

### 7.2 数据目录

- 资产数据存储位置：`knowledge/assets/`
- 文件命名格式：`{corp_id}_assets.json`
- 自动创建：首次使用时自动创建目录和文件
