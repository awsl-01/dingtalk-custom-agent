# PPT生成方法对比

## 两种方法的区别

### 方法1：设计基因提取法 (`template_design_ppt.py`)

**原理**：
1. 从模板中提取设计参数（颜色、字体、装饰等）
2. 使用空白布局创建新幻灯片
3. 根据提取的参数手动创建所有元素

**优点**：
- 可以创建多种版式类型（章节页、图文页、引用页等）
- 可以自动调整布局多样性
- 可以检查文字溢出

**缺点**：
- 无法完全复制模板的视觉效果
- 需要手动创建所有元素
- 可能与模板有细微差异

**适用场景**：
- 需要多种版式类型的PPT
- 需要自动布局调整的场景
- 模板结构简单（只有颜色和字体）

---

### 方法2：模板复制法 (`template_based_ppt.py`)

**原理**：
1. 直接复制模板幻灯片
2. 修改现有文本框的内容
3. 保留所有格式和样式

**优点**：
- 完全保留模板的视觉效果
- 字体、颜色、装饰完全一致
- 实现简单，不易出错

**缺点**：
- 版式类型固定（只能使用模板中已有的版式）
- 无法自动创建新布局
- 需要模板中有对应的幻灯片

**适用场景**：
- 模板中已有封面页和目录页
- 需要完全一致的视觉效果
- 不需要复杂的版式变化

---

## 推荐使用方法

### 对于你的模板（只有封面页和目录页）

**推荐使用方法2：模板复制法**

原因：
1. 你的模板只有2张幻灯片（封面页和目录页）
2. 模板不是在母版中设置的，只是普通幻灯片
3. 需要完全保留模板的格式和样式

### 使用示例

```python
from agent.template_based_ppt import generate_template_based_ppt

output_path, title = generate_template_based_ppt(
    user_message="生成一个从百草园到三味书屋适合初中的语文课件PPT",
    template_path="templates/default_template.pptx",
    output_dir="projects"
)
```

### 生成效果

**封面页**：
- 完全复制模板封面页的格式
- 只修改标题和副标题文本

**内容页**：
- 完全复制模板目录页的格式
- 修改标题和项目符号列表
- 每页都保持相同的布局和样式

**结束页**：
- 复制封面页格式
- 将标题改为"谢谢观看"

---

## 如何选择

### 选择方法1（设计基因提取）当：
- 需要多种版式类型（章节页、图文页、引用页等）
- 需要自动调整布局多样性
- 模板结构复杂，有多种版式

### 选择方法2（模板复制）当：
- 模板只有简单的封面页和目录页
- 需要完全一致的视觉效果
- 不需要复杂的版式变化
- 模板不是在母版中设置的
