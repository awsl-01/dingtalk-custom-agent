# 基于模板设计基因的PPT生成器 - 完成总结

## 功能概述

本模块能够从用户提供的PPT模板中提取"设计基因"，并基于这些基因生成风格统一的完整PPT。

## 已实现的功能

### 1. 设计基因提取 ✓

从模板文件中自动提取以下设计元素：
- **封面页**：背景颜色（用于章节页和结束页）
- **目录页**：背景颜色（用于正文页）
- **标题样式**：字体、字号、颜色、加粗/倾斜属性
- **正文样式**：字体、字号、颜色、行距
- **项目符号**：样式、缩进
- **装饰元素**：装饰线颜色、位置、高度
- **页码**：位置、格式

### 2. 动态创建风格一致的版式 ✓

根据提取的设计基因，为每张内容幻灯片动态创建：
- **背景**：复用封面/目录页的纯色背景
- **标题框**：位置、尺寸保持统一，样式与模板一致
- **正文框**：根据基因中的正文字体样式创建
- **装饰元素**：保留细线或色块维持视觉连贯

### 3. 版式规划与类型分配 ✓

支持多种版式类型，避免单调：
- **封面页**：深色背景、大标题
- **目录页**：项目符号列表
- **章节页**：深色背景+半透明遮罩+大标题居中
- **正文页**：浅色背景+标题+要点列表
- **图文页**：左文右图布局
- **引用/金句页**：居中大号文本
- **结束页**：深色背景

**强制要求**：连续三页不能使用完全相同的布局类型。

### 4. 防止文字溢出和排版混乱 ✓

- **容量预估**：根据正文字号、行距和文本框宽度计算可容纳行数
- **自动拆分**：内容过多时自动拆分到新幻灯片
- **最小字号**：启用自动调整时字号不低于14pt
- **溢出检查**：生成每页后立即检查是否溢出

### 5. 最终自查清单 ✓

生成完毕前自动输出检查结果：
```
第X页：版式类型【正文/章节等】，标题字号XX，正文字号XX，是否溢出：否
```

## 使用方法

### 1. 准备模板文件

将模板文件（.pptx格式）放入 `templates` 目录。

**模板要求**：
- 至少包含2张幻灯片：封面页和目录页
- 封面页应包含标题和副标题
- 目录页应包含标题和项目符号列表

### 2. 使用钉钉机器人

向钉钉机器人发送包含以下关键词的消息：
- "模板"
- "风格统一"
- "设计基因"
- "套用模板"

**示例**：
```
使用模板生成一个关于人工智能发展趋势的PPT
```

### 3. 使用Python代码

```python
from agent.template_design_ppt import generate_template_ppt

# 生成PPT
output_path, title = generate_template_ppt(
    user_message="生成一个关于人工智能发展趋势的PPT",
    template_path="templates/default_template.pptx",
    output_dir="projects"
)

print(f"PPT已生成: {output_path}")
```

## 技术实现

### 核心类

1. **DesignGene**：存储设计基因的数据类
2. **TemplateDesignExtractor**：从模板中提取设计基因
3. **SlideBuilder**：根据设计基因构建幻灯片
4. **TextOverflowChecker**：检查并防止文字溢出
5. **LayoutDiversityChecker**：确保版式多样性
6. **TemplateDesignPPTGenerator**：主生成器类

### 关键修复

1. **设计基因提取**：正确区分封面页和目录页
2. **字体大小检测**：使用 `run = p.add_run()` 方式设置文本
3. **半透明遮罩**：添加异常处理和备选方案
4. **负间距问题**：移除不支持的负间距设置

## 注意事项

1. **模板质量**：模板的设计质量直接影响生成结果
2. **字体兼容**：确保模板使用的字体在目标环境可用
3. **内容长度**：过长的内容会被自动拆分或精简
4. **API配置**：需要配置OpenAI兼容的API密钥
5. **版本要求**：需要python-pptx>=0.6.21
