# 基于模板设计基因的PPT生成器

## 功能概述

本模块能够从用户提供的PPT模板中提取"设计基因"，并基于这些基因生成风格统一的完整PPT。

## 核心特性

### 1. 设计基因提取

从模板文件中自动提取以下设计元素：
- 背景颜色或背景图片
- 标题的字体、字号、颜色、加粗/倾斜属性
- 正文的字体、字号、颜色、行距
- 目录页的项目符号样式、缩进
- 页脚/页码的位置、格式
- 装饰元素（色块、线条）

### 2. 动态创建风格一致的版式

根据提取的设计基因，为每张内容幻灯片动态创建：
- **背景**：复用封面/目录页的纯色背景
- **标题框**：位置、尺寸保持统一，样式与模板一致
- **正文框**：根据基因中的正文字体样式创建
- **装饰元素**：保留细线或色块维持视觉连贯

### 3. 版式规划与类型分配

支持多种版式类型，避免单调：
- **章节页**：全图背景+半透明遮罩+大标题
- **正文页**：标准标题+要点列表
- **图文页**：左文右图，图片用占位符
- **引用/金句页**：居中大号文本
- **结束页**：类似封面的色调

**强制要求**：连续三页不能使用完全相同的布局类型。

### 4. 防止文字溢出和排版混乱

- **容量预估**：根据正文字号、行距和文本框宽度计算可容纳行数
- **自动拆分**：内容过多时自动拆分到新幻灯片
- **最小字号**：启用自动调整时字号不低于14pt
- **溢出检查**：生成每页后立即检查是否溢出

### 5. 最终自查清单

生成完毕前自动输出检查结果：
```
第X页：版式类型【正文/章节等】，标题字号XX，正文字号XX，是否溢出：否
```

## 使用方法

### 1. 准备模板文件

将模板文件（.pptx格式）放入 `templates` 目录。

**模板要求**：
- 至少包含2张幻灯片：封面页和目录页
- 母版中可以没有其他版式
- 封面页应包含标题和副标题
- 目录页应包含标题和项目符号列表

### 2. 使用钉钉机器人

向钉钉机器人发送包含以下关键词的消息：
- "模板"
- "风格统一"
- "设计基因"
- "套用模板"

**示例**：
```
使用模板生成一个关于人工智能发展趋势的PPT
```

### 3. 使用Python代码

```python
from agent.template_design_ppt import generate_template_ppt

# 生成PPT
output_path, title = generate_template_ppt(
    user_message="生成一个关于人工智能发展趋势的PPT",
    template_path="templates/default_template.pptx",
    output_dir="projects"
)

print(f"PPT已生成: {output_path}")
```

## 技术实现

### DesignGene 数据类

存储从模板中提取的所有设计参数：
- 背景相关：颜色、图片、类型
- 标题样式：字体、字号、颜色、加粗、对齐
- 正文样式：字体、字号、颜色、行距
- 装饰元素：强调色、装饰线、色块
- 布局参数：边距、区域位置

### SlideBuilder 幻灯片构建器

根据设计基因构建不同类型的幻灯片：
- `_build_content_slide()`: 正文页
- `_build_chapter_slide()`: 章节页
- `_build_image_text_slide()`: 图文页
- `_build_quote_slide()`: 引用页

### TextOverflowChecker 溢出检查器

- `check_overflow()`: 检查文本是否会溢出
- `split_content()`: 将内容拆分到多张幻灯片

### LayoutDiversityChecker 多样性检查器

- `check_and_fix_layout()`: 检查并修正版式，确保无三页连用同一种结构

## 注意事项

1. **模板质量**：模板的设计质量直接影响生成结果
2. **字体兼容**：确保模板使用的字体在目标环境可用
3. **内容长度**：过长的内容会被自动拆分或精简
4. **API配置**：需要配置OpenAI兼容的API密钥
