# 模板PPT生成器 - 最终总结

## 问题分析

你提供的模板只有2张幻灯片（封面页和目录页），并且不是在母版中设置的。这导致：

1. **原方法（设计基因提取）**：无法完全复制模板的视觉效果
2. **新方法（模板复制）**：可以直接复制模板幻灯片，保留所有格式

## 解决方案

### 新增模块：`agent/template_based_ppt.py`

**原理**：
- 直接复制模板幻灯片
- 修改现有文本框的内容
- 保留所有格式和样式

**优点**：
- 完全保留模板的视觉效果
- 字体、颜色、装饰完全一致
- 适用于普通幻灯片（非母版版式）

### 使用方法

**通过钉钉机器人**：
```
生成一个从百草园到三味书屋适合初中的语文课件PPT模板风格统一
```

**通过Python代码**：
```python
from agent.template_based_ppt import generate_template_based_ppt

output_path, title = generate_template_based_ppt(
    user_message="生成一个从百草园到三味书屋适合初中的语文课件PPT",
    template_path="templates/default_template.pptx",
    output_dir="projects"
)
```

## 如何使用

### 1. 准备模板
将你的模板文件放入 `templates` 目录。

### 2. 发送消息
向钉钉机器人发送：
```
生成一个[主题]的PPT模板风格统一
```

### 3. 接收PPT
系统会自动生成风格统一的PPT并发送给你。

---

## 注意事项

1. **模板格式**：模板必须是普通幻灯片，不是母版版式
2. **幻灯片数量**：模板至少需要2张幻灯片（封面页和目录页）
3. **文本框**：模板中的文本框会被识别并修改
4. **API配置**：需要配置OpenAI兼容的API密钥

---

## 技术支持

如有问题，请检查：
1. 模板文件是否在 `templates` 目录中
2. 模板格式是否正确（至少2张幻灯片）
3. API配置是否正确（.env文件）
4. 依赖包是否安装（python-pptx, openai等）
