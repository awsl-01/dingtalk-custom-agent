# 模板PPT生成器使用说明

## 快速开始

### 1. 准备模板文件

将你的模板文件（.pptx格式）放入 `templates` 目录。

**模板要求**：
- 至少包含2张幻灯片：封面页和目录页
- 可以是普通幻灯片，不需要在母版中设置
- 封面页应包含标题和副标题文本框
- 目录页应包含标题和项目符号列表

### 2. 使用钉钉机器人

向钉钉机器人发送消息，格式如下：

**基本格式**：
```
生成一个[主题]的PPT模板风格统一
```

**示例**：
```
生成一个从百草园到三味书屋适合初中的语文课件PPT模板风格统一
```

**触发关键词**：
- `模板` 或 `风格统一` - 触发模板PPT生成
- `ppt` 或 `PPT` - 触发PPT生成
- `初中`、`高中`、`小学` 等 - 标识为教育PPT

### 3. 使用Python代码

```python
from agent.template_based_ppt import generate_template_based_ppt

# 生成PPT
output_path, title = generate_template_based_ppt(
    user_message="生成一个从百草园到三味书屋适合初中的语文课件PPT",
    template_path="templates/default_template.pptx",
    output_dir="projects"
)

print(f"PPT已生成: {output_path}")
```

## 常见问题

### Q: 为什么生成的PPT格式与模板不一致？

A: 请确保：
1. 模板文件在 `templates` 目录中
2. 模板至少包含2张幻灯片（封面页和目录页）
3. 模板是普通幻灯片，不是母版版式

### Q: 如何自定义模板？

A: 创建一个PPT文件，包含：
1. 第1页：封面页（包含标题和副标题）
2. 第2页：目录页（包含标题和项目符号列表）

然后将文件放入 `templates` 目录。

### Q: 支持哪些关键词？

A: 支持以下关键词：
- PPT相关：`ppt`、`PPT`、`幻灯片`、`演示文稿`
- 模板相关：`模板`、`风格统一`、`设计基因`、`套用模板`
- 年级相关：`小学`、`初中`、`高中`、`大学`
- 学科相关：`语文`、`数学`、`英语`、`物理` 等
