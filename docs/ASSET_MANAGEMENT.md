# 资产管理功能使用说明

## 功能概述

学校资产管理功能支持通过钉钉机器人进行资产的录入、查询、借用、归还、统计和模板导入。

## 使用方式

### 1. 录入资产

**格式**：`录入资产 [名称] [分类] [位置]`

**示例**：
```
录入资产 投影仪 教学设备 301教室
添加资产 电脑 办公设备 办公室
新增资产 桌椅 家具 201教室
```

**返回信息**：
```
[成功] 资产录入成功！

[资产信息]
- 编号：AST20260618001
- 名称：投影仪
- 分类：教学设备
- 位置：301教室
- 状态：在用
- 录入时间：2026-06-18 10:30:00
```

### 2. 查询资产

**格式**：`查询资产 [名称/编号]`

**示例**：
```
查询资产 投影仪          # 按名称查询
查询资产 AST20260618001  # 按编号查询
资产列表                  # 查看所有资产
```

**按名称查询返回**：
```
[查询结果] 查询到 3 条资产记录：

- AST20260618001 | 投影仪 | 在用 | 301教室
- AST20260618002 | 投影仪 | 在用 | 302教室
- AST20260618003 | 投影仪 | 借用中 | 303教室
```

**按编号查询返回**：
```
[资产详情]

- 编号：AST20260618001
- 名称：投影仪
- 分类：教学设备
- 位置：301教室
- 状态：在用
- 负责人：张老师
- 采购日期：2026-06-18
```

### 3. 借用资产

**格式**：`借用资产 [编号] [借用人]`

**示例**：
```
借用资产 AST20260618001 李老师
```

**返回信息**：
```
[成功] 资产借用成功！

[借用信息]
- 资产编号：AST20260618001
- 资产名称：投影仪
- 借用人：李老师
- 借用时间：2026-06-18 14:00:00

[提示] 请记得及时归还！
```

### 4. 归还资产

**格式**：`归还资产 [编号]`

**示例**：
```
归还资产 AST20260618001
```

**返回信息**：
```
[成功] 资产归还成功！

[归还信息]
- 资产编号：AST20260618001
- 资产名称：投影仪
- 归还时间：2026-06-18 16:00:00

感谢使用！
```

### 5. 资产统计

**格式**：`资产统计`

**返回信息**：
```
[资产统计报告]

[总资产数] 25 台

[状态分布]
- 在用：20 台
- 借用中：3 台
- 维修中：2 台

[分类分布]
- 教学设备：15 台
- 办公设备：5 台
- 家具：5 台

[位置分布（前5）]
- 301教室：5 台
- 302教室：4 台
- 办公室：3 台
```

### 6. 下载资产录入模板

**格式**：`资产录入模板`

**返回信息**：
```
[资产录入模板已生成]

[模板字段说明]
- 资产名称：必填
- 资产分类：教学设备/办公设备/家具/教材/其他
- 存放位置：必填（如：301教室、办公室）
- 负责人：选填
- 采购日期：选填（格式：2026-06-18）
- 资产描述：选填

[使用说明]
1. 下载并打开模板文件
2. 按照字段说明填写资产信息
3. 保存文件后上传
4. 发送「批量导入资产」进行导入
```

**说明**：系统会发送一个 Excel 或 CSV 模板文件，用户可以下载后填写资产信息，然后上传进行批量导入。

### 7. 下载资产盘点模板

**格式**：`资产盘点模板`

**返回信息**：
```
[资产盘点模板已生成]

[模板字段说明]
- 资产编号：必填（AST开头的编号）
- 资产名称：必填
- 实际位置：必填（盘点时的实际位置）
- 资产状态：在用/闲置/维修中/报废
- 盘点人：必填
- 盘点日期：必填（格式：2026-06-18）
- 备注：选填

[使用说明]
1. 先使用「资产列表」导出当前资产清单
2. 下载盘点模板并填写实际盘点结果
3. 上传填写好的模板文件
4. 系统将自动更新资产状态
```

### 8. 批量导入资产

**格式**：批量导入资产（需先上传填写好的模板文件）

**操作流程**：
1. 发送「资产录入模板」下载模板
2. 按照模板填写资产信息
3. 上传填写好的文件
4. 发送「批量导入资产」完成导入

**返回信息**：
```
[批量导入完成]

[导入结果]
- 成功导入：10 条资产记录
- 数据已保存到知识库

[提示] 使用「资产列表」查看导入的资产
```

## 资产编号规则

- 格式：`AST` + `日期(8位)` + `序号(3位)`
- 示例：`AST20260618001`
- 同一天最多支持 999 个资产

## 支持的资产分类

| 分类 | 包含设备 |
|------|----------|
| 教学设备 | 投影仪、电脑、白板、音响、麦克风 |
| 办公设备 | 打印机、复印机、扫描仪、电脑 |
| 家具 | 桌椅、课桌、椅子、柜子 |
| 教材 | 教材、课本、教辅 |
| 其他 | 未分类的资产 |

## 资产状态说明

| 状态 | 说明 |
|------|------|
| 在用 | 资产正常使用中，可被借用 |
| 借用中 | 资产已被借出，需归还后才能再次借用 |
| 维修中 | 资产正在维修，暂不可用 |
| 报废 | 资产已报废，不再使用 |

## 数据存储

- 存储位置：`knowledge/assets/{corp_id}_assets.json`
- 存储格式：JSON
- 数据隔离：按 corp_id（企业ID）隔离不同学校的数据

## 注意事项

1. **资产编号**：系统自动生成，无需手动输入
2. **借用限制**：只有状态为「在用」的资产才能被借用
3. **归还检查**：只有状态为「借用中」的资产才能归还
4. **数据安全**：建议定期备份 `knowledge/assets/` 目录

## 技术架构

```
agent/skills/
├── asset_skill.py          # 资产管理技能主模块
└── asset_storage.py        # 资产数据存储模块
```

- **技能注册**：自动加载，无需修改 `main.py`
- **数据存储**：JSON 文件，按 corp_id 隔离
- **意图识别**：关键词匹配，支持多种表述方式
